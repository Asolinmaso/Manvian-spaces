# Manvian Spaces - M SPACE

A Next.js 15 project with Tailwind CSS for the Manvian Spaces co-working website.

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 3**
- **TypeScript**

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Project Structure

- `src/app/` - App Router pages and layout
- `public/images/` - Static images (Logo, Hero, Footer, etc.)
